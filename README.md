| Dossier | Rôle |
|--------|------|
| `backend/` | API Node.js + Express |
| `frontend/client/` | Application React — parcours client (démo) |
| `frontend/admin/` | Application React — back-office analyste |
| `docs/` | Cahier des charges, MERISE, UML, plan de travail |
